# Основы React
## Создание карты поиска
В папке ```src``` создаем файл ```MovieCard.jsx```  
В этом файлк пишем:
```jsx
export function MovieCard() {
  return <div>MovieCard</div>
}
```
добавим входящие данные в эту функцию:
```jsx
export function MovieCard({image, rating}) {
  return <div>MovieCard</div>
}
```

Добавим поле картинки и рейтинга во фрагмент
```jsx
export function MovieCard({image, rating}) {
  return <div>
    <div>
      IMDb rating^ {rating}
    </div>
    <img src={image} alt="" />
  </div>
}
```

Далее:  
1. Удалим файл ```src/App.css``` 
1. изменим файл ```src/App.tsx```
```jsx
import { MovieCard } from "./MovieCard";

function App() {
  return (
    <>
      <MovieCard image="/1920x.webp" rating={8.8} />
    </>
  );
}

export default App;

```
Добавляем ширину в ```MovieCard.jsx```
```jsx
...
      <div>IMDb rating: {rating}</div>
      <img src={image} alt="" width={200}/>
    </div>
...
```
# Установка Tailwind в наш проект
Tailwind - нужен для бымтрого и простого дизайна стилей
* В Terminal прописываем комеду ```npm install tailwindcss @tailwindcss/vite```
* После переходим  файл ```vite.config.js``` и в нем подключаем наш **Tailwind**
* В строке ```plugins: [react()],``` после ```react()``` через запятую пишем ```tailwindcss()```
* И добавляем импорт ```import tailwindcss from "@tailwindcss/vite";```

### Итоговый файл ```vite.config.js```
```jsx
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from "@tailwindcss/vite";

// https://vite.dev/config/
export default defineConfig({
  plugins: [react(), tailwindcss()],
})

```
* И в index.css импортируем **Tailwind** ```@import "tailwindcss";```
