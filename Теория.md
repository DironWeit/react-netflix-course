## Создание карты поиска
В папке ```src``` создаем файл ```MovieCard.jsx```  
В этом файлк пишем:
```jsx
export function MovieCard() {
  return <div>MovieCard</div>
}
```
добавим входящие данные в эту функцию:
```jsx
export function MovieCard({image, rating}) {
  return <div>MovieCard</div>
}
```

Добавим поле картинки и рейтинга во фрагмент
```jsx
export function MovieCard({image, rating}) {
  return <div>
    <div>
      IMDb rating^ {rating}
    </div>
    <img src={image} alt="" />
  </div>
}
```

Далее:  
1. Удалим файл ```src/App.css``` 
1. изменим файл ```src/App.tsx```
```jsx
import { MovieCard } from "./MovieCard";

function App() {
  return (
    <>
      <MovieCard image="/1920x.webp" rating={8.8} />
    </>
  );
}

export default App;

```
Добавляем ширину в ```MovieCard.jsx```
```jsx
...
      <div>IMDb rating: {rating}</div>
      <img src={image} alt="" width={200}/>
    </div>
...
```
# Установка Tailwind в наш проект
Tailwind - нужен для бымтрого и простого дизайна стилей
* В Terminal прописываем комеду ```npm install tailwindcss @tailwindcss/vite```
* После переходим  файл ```vite.config.js``` и в нем подключаем наш **Tailwind**
* В строке ```plugins: [react()],``` после ```react()``` через запятую пишем ```tailwindcss()```
* И добавляем импорт ```import tailwindcss from "@tailwindcss/vite";```

### Итоговый файл ```vite.config.js```
```jsx
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from "@tailwindcss/vite";

// https://vite.dev/config/
export default defineConfig({
  plugins: [react(), tailwindcss()],
})

```
* И в index.css импортируем **Tailwind** ```@import "tailwindcss";```

# Создание кнопки избранного
Создадим кнопку для добаление в избранное на карточке фильма  
### Создадим новй файл ```scr\FavoriteButton.jsx``` и в нем пропишем:
> ```useState``` - это хук для хранения сотояния и возращает каотеж  
>> ```кортеж``` - как массив, только отдает конкретное число элементов  

```jsx 
const [isFavorite, setIsFaforite] = useState(false)
//- создание useState, где
//                isFavorite- переменная
//                setIsFaforite - функция изменения состояния
```

И в ```return``` добавим это:
```jsx
return (<button onClick={() => setIsFaforite(!isFavorite)}>  // тут мы меняем состояние на противоположное
    {isFavorite ? '❤️' :'😍'} // выставляем иконку в зависемости от состояния
  </button>
  )
```

### Подключим к файлу ```MovieCard.jsx```
Просто в главном ```div``` добавим строку ```<FavoriteButton/>```  
Накинем стили и получится так:
```jsx 
import { FavoriteButton } from "./FavoriteButton";

export function MovieCard({ image, rating }) {
  return (
    <div className="relative w-[200px] rounded-2xl overflow-hidden bg-neutral-900 shadow-lg hover:scale-105 transition-transform will-change-transform duration-300">
      <img
        src={image}
        alt="Movie Poster"
        className="w-full h-auto object-cover" />
      
      {/* Кнопка избранного */}
      <div className="absolute top-2 right-2 z-10">
        <FavoriteButton/>
      </div>
      <div className="absolute bottom-0 left-0 w-full bg-gragient-to-t from-black/80 to-transparent p-2 text-sm text-white font-semibold">
        IMDb: {rating}
      </div>
    </div>
  )
}
```
